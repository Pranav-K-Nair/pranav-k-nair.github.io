Project Showcase

LLM Cold Email Generator

Ever had anxiety while staring at a blank email draft, unsure of how to perfectly pitch yourself for that dream job? Well, your writers block ends here! This project harnesses the power of AI to transform the tedious task of crafting personalized cold emails into a breeze. Using LangChain and state-of-the-art LLMs like Llama and OpenAI GPT, I built a system that scrapes job postings, analyzes your resume, and generates tailored application emails.

But here's where it gets really cool: the system doesn't just throw together generic templates. It uses ChromaDB vector database to perform semantic search, intelligently matching your portfolio projects with the job's specific requirements. Need to highlight your machine learning experience for a data science role? Done. Want to emphasize your web development chops for a full-stack position? Already on it. The AI automatically identifies which of your projects deserves the spotlight for each unique application.

It's all wrapped up in a sleek Streamlit web application that handles everything, just paste a job posting URL, upload your resume, and watch as the magic of prompt engineering transforms raw data into a polished, personalized email ready to land in a recruiter's inbox. Because why spend hours crafting the perfect email when you can teach an AI to do it in seconds?

Fraud Detection System

DON'T REDEEM THE GIFT CARD. But more importantly, how do you (or a bank) tell whether there's a fraudster scamming people out of their money? Into this project, I dove deep into the world of financial fraud detection to build a machine learning system that can spot suspicious transactions before they empty someone's bank account. Starting with exploratory data analysis and hypothesis testing, I engineered features and preprocessed the dataset to handle both numerical and categorical data. After establishing a baseline, I put multiple ML models to the test, comparing their performance through cross-validation. XGBoost emerged as the champion, so I fine-tuned it using hyperparameter optimization and deployed the final model through a Streamlit web application. Because sometimes the best defense against fraud is a good algorithm that never sleeps!

Parallel Processing Image Convolution

This project explores different approaches to optimizing image convolution performance using a 5x5 Gaussian filter. I implemented and compared serial execution against various parallel processing techniques, including OpenMP and AVX2 for CPU parallelization, and CUDA with both Global Memory and Shared Memory approaches for GPU acceleration. The project features an interactive GUI that allows users to test each algorithm implementation and visualize performance results through integrated graphs, providing a comprehensive tool for understanding the performance benefits of different parallelization strategies in image processing applications.
